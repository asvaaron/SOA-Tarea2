## Ejecutar Programa

Para ejecutar el progama rastreador es necesario utilizar la siguiente estructura:


```bash
./main [opciones rastreador] Prog [opciones de Prog]
```

Siendo las opciones del rastreador las banderas -v y -V. La bandera -v despliega un mensaje cada vez que detecte un System Call, mientras que -V hace una pausa hasta que el usuario presione cualquier tecla para continuar la ejecuci√≥n. 

En el caso particular de que no venga ninguna bandera -v o -V, se asume por default que el usuario introdujo la bandera -v.


### Ejemplos:

```bash 
./main ls -al
```

```bash 
./main -v ls -al
```

```bash 
./main -V ls -al
```
